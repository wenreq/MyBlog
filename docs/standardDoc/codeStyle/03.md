### 一些规范

- space2。
- 标签必须合法且闭合、嵌套正确，标签名需小写。
- 标签语义化，尽量避免整个页面都是 `div` 来布局，不便于理解和维护。
- 除非有特定要求，禁止随意使用 `id` 来定义元素样式。
- 使用 `a` 签时加上 `title` 属性
- 使用 `img` 标签时加上 `alt` 属性
