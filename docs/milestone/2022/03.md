2. 网站引导页引入字体文件大，加载慢的优化。font-spider 字体压缩器，让网页自由引入中文字体成为可能。

```shell
npm install font-spider -g
```

运行 font-spider 命令

```shell
font-spider  *.html
```

命令执行完后，字体文件下多出来以下文件（.font-spider文件夹下是原始的ttf文件）
[链接](https://www.jianshu.com/p/3a2c7272269d)
